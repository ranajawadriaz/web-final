Web Programming - Quiz 2
Version A


Question: Create a small counter app using the provided HTML snippet.    


Requirements: Must have a minus button, plus button, reset button, a textbox showing the current value, and a div containing the step value.    

Behavior: When '+' is clicked, add the step value. When '-' is clicked, subtract the step value. When 'Reset' is clicked, set the value to 0.    


Rules: Step value must be read from the div (not hardcoded), value should update live, value cannot drop below 0, and you must use JavaScript.    

Version B


Question: Create a small temperature app using the provided HTML snippet.    


Requirements: Must have a heat button, cool button, reset button, a textbox showing the current temperature, and a div containing the maximum allowed temperature.    


Behavior: When 'Heat' is clicked, increase temperature by 2. When 'Cool' is clicked, decrease temperature by 2. When 'Reset' is clicked, set temperature to 20 (default).    


Rules: Temperature must not exceed the maximum, must not go below 0, updates must happen live, and you must use JavaScript only.    

Version C


Question: Create a transaction counter app using the provided HTML snippet.    


Requirements: Must have a deposit button, withdraw button, reset button, a div containing the transaction amount, and a textbox showing the current balance.    

Behavior: When 'Deposit' is clicked, add the transaction amount to the balance. When 'Withdraw' is clicked, subtract the transaction amount. When 'Reset' is clicked, set the balance to 0.    


Rules: Balance must NEVER go below 0, the textbox must be updated immediately, use JavaScript only, and the code should still work if the amount changes in the HTML.
