Adding and Removing Classes JS
const element = document.querySelector(".box");
element.classList.add("active");
Adds the class activeDoes NOT remove existing classes
Remove a Class
element.classList.remove("active");
Removes only the class active Other classes remain unchanged
Toggle a Class
element.classList.toggle("active");
Adds the class if not present Removes the class if already present
Check if Class Exists
element.classList.contains("active");
Returns true or false
